## (3) Ressources à RECLASSER par rapport au périmètre initial
- **Hélium, silicium métal, charbon à coke : DÉJÀ CRMA-listés** → traiter comme **CRMA (cadre existant), pas comme ressources « nouvelles »**. (Hélium = Critique/Annexe II ; silicium métal = Stratégique + Critique ; charbon à coke = Critique/Annexe II.)
- **Uranium : PAS une matière CRMA** → ne pas le modéliser sous CRMA ; il relève du **régime Euratom / ESA**. Cadre légal distinct.
- **Hydrogène : à cheval sur trois cadres** → **PAS matière première CRMA**, mais (a) **RFNBO sous RED III**, et (b) **bien couvert CBAM**. Classer sous énergie/CBAM, pas CRMA.
- **Charbon à coke : CRMA-critique mais PAS bien CBAM** → garder son classement CRMA ; ne pas le taguer CBAM (carbone compté en émissions incorporées dans l'acier).
- **Xénon, krypton, argon et (comme matière première) hydrogène : hors périmètre CRMA** → si MODULE 2 groupait tous les gaz industriels comme « critiques », **seul l'hélium** est CRMA-critique ; les autres sont des **gaz industriels non-CRMA** (régis via REACH/CLP, et — pour GNL/gaz — le régime sécurité d'appro. gaz).
- **ESRS E5 (pas E1) est le standard d'information d'ancrage** pour le récit dépendance-ressources / matières-critiques de ce module ; E2 rattaché aux angles substances REACH/pollution ; ESRS 1/2 régissent le screening IRO de double matérialité. **Tenir compte de la révision ESRS simplifiés en attente** avant de figer des champs d'information.
